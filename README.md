# MERN E-Commerce Platform

A full-stack e-commerce web application developed using the MERN stack, focusing on real-world business workflows and scalable architecture.

## Overview
This project demonstrates the design and implementation of a complete e-commerce system, including user management, product handling, and order workflows. It follows a modular structure suitable for real-world full-stack applications.

## Key Features
- User authentication and authorization
- Product listing and management
- Shopping cart and order processing
- Admin dashboard for managing users and orders

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## Skills Demonstrated
- Full-stack web development
- RESTful API design
- Database modeling and CRUD operations
- Authentication and role-based access control
