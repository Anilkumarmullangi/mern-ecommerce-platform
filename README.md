# MERN E-Commerce Platform

A full-stack e-commerce web application built using MongoDB, Express.js, React.js, and Node.js.

## Features
- User authentication and authorization (JWT)
- Product listing and management
- Shopping cart and order processing
- Admin dashboard

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
